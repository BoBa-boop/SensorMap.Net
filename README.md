## Назначение

Приложение предназначено для промышленных предприятий и позволяет:

- Располагать датчики и устройства (ПЛК, контроллеры) непосредственно на изображениях схем оборудования
- Вести справочники датчиков, устройств, их типов и характеристик
- Хранить полезную информацию: документацию, фото реального расположения, электрические схемы
- Организовать структуру по участкам и оборудованию

## Возможности

### Карта оборудования
- Интерактивный редактор схем с поддержкой перетаскивания (drag-and-drop)
- Размещение датчиков и устройств на фоновом изображении схемы
- Масштабирование (колёсико мыши) и перемещение (правая кнопка мыши)
- Выделение нескольких объектов, копирование/вставка/удаление
- Отмена и повтор действий (Ctrl+Z / Ctrl+Y)
- Умное размещение адресных меток с избеганием перекрытий
- Фото реального расположения для каждого размещённого объекта

### Управление данными
- Участки (секторы) с привязанным оборудованием
- Датчики и устройства с типами, характеристиками и изображениями
- Привязка ПЛК-адресов к датчикам на карте
- Прикрепление документов и файлов к механизациям

### Интерфейс
- Два режима: просмотр (только чтение) и редактор (с авторизацией)
- Иерархическое дерево навигации: Участки → Оборудование → Датчики / Устройства
- CRUD-редактор с табличным представлением для массового управления данными
- Настройки: смена пароля, загрузка другой базы данных, резервное копирование

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Целевая платформа | .NET 9, Windows (WPF) |
| UI-библиотека | HandyControl 3.5.1 |
| MVVM | ReactiveUI 21, CommunityToolkit.Mvvm 8.4 |
| ORM | Entity Framework Core 9 |
| База данных | SQLite (`AppDataBase.db`) |
| Логирование | NLog |
| Сериализация | Newtonsoft.Json |

## Структура проекта

```
SensorMap/
├── Model/              # Сущности: Sector, Mechanism, Sensor, Device и др.
├── ViewModel/          # ViewModel'ы (MVVM-паттерн)
├── View/               # XAML-окна и страницы
├── EF/                 # DbContext (AppDBContext)
├── Services/           # Сервисы: навигация, данные, файлы, авторизация
├── Interfaces/         # Интерфейсы сервисов
├── CustomControls/     # Пользовательские элементы управления (карта, датчик, устройство)
├── Converters/         # WPF-конвертеры значений
├── Migrations/         # Миграции EF Core
├── Resources/          # Изображения, курсоры, стили, SVG-иконки
└── Themes/             # Шаблоны контролов (Generic.xaml)
```

При первом запуске автоматически создастся файл базы данных `AppDataBase.db`.
